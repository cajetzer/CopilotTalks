Usage: copilot init [options]

Initialize Copilot instructions for this repository.

Analyzes your codebase using a restricted set of read-only tools and generates a
`.github/copilot-instructions.md` file with contextual guidance for Copilot,
including build and test commands, coding conventions, project structure, and
technology stack.

The generated instructions help Copilot understand your project's conventions so
it can produce more relevant code suggestions and follow your team's practices.

For more information, see:
https://docs.github.com/copilot/how-tos/configure-custom-instructions/add-repository-instructions

Options:
  -h, --help  display help for command

Examples:
  # Initialize instructions in the current repository
  $ copilot init

Learn More:
  Use `copilot <command> --help` for more information about a command.
  Read the documentation at https://docs.github.com/copilot/how-tos/copilot-cli
